## Technical Specification
- **Database** : Postgresql
- **Backend** : SpringFramework
    - openjdk 1.8
    - maven
- **Frontend** : Vue.js
    - ECMAScript 2015
    - webpack

## Development Environment
- **OS** : Cent OS
- **Based On** : Docker, Nginx
- **IDE** : code-server
- **Query** : pgadmin4
